CSS Introduction

CSS is a complex language that packs quite a bit of power.

It allows us to add layout and design to our pages, and it allows us to share those styles from element to element and page to page.

#food { background: green; } p { background: orange; }

Box Model

According to the box model concept, every element on a page is a rectangular box and may have width, height, padding, borders, and margins.

Every element is a rectangular box, and there are several properties that determine the size of that box. The core of the box is defined by the width and height of an element, which may be determined by the display property, by the contents of the element, or by specified width and height properties. padding and then border expand the dimensions of the box outward from the elementâ€™s width and height. Lastly, any margin we have specified will follow the border.

div { border: 6px solid #949599; height: 100px; margin: 20px; padding: 20px; width: 400px; }

Colors

The only property we need to set the color of text is the color property. The color property accepts one color value, but in many different formats.

Hexadecimal values are the most prevalent, as they provide the most control with the least amount of effort.

p { background: orange; font-size: 24px; } p { background: green; }

Typography

CSS offers a lot of different properties for editing the look and feel of text on a page. These properties fit into two categories: font-based properties and text-based properties.

Font family, font size, font style, font variant, font weight & line height are just a few examples of how typography can be altered using CSS.

body { font-family: "Helvetica Neue", Helvetica, Arial, sans-serif; }

Grid

Flexible grids are great for positioning content.

Row and column size are altered and justified to create a layout.

.container { width: 538px; } section, aside { margin: 10px; } section { float: left; width: 340px; } aside { float: right; width: 158px; }
